How does this work?

Is my commit message useful uses machine learning to determine how good and well written a commit message is. The ML model is trained from a large set of commit messages from well-known open-source projects.
Above you can rate the messages in the data set and improve the ML model. The model gets retrained every 5 minutes. Clicks on I don't know won't influnce the training process, it's just for sorting out bad commit messages.
